✨ Key Features
🔍 Search any location worldwide
🌡️ Celsius/Fahrenheit toggle
🌙 Dark/Light mode
📍 Auto-detect your location
📊 5-day forecast
📱 Installable PWA

🛠️ Built With
⚛️ React
🌤️ OpenWeather API
🎨 CSS Modules
🔌 Axios

🚀 Quick Start

bash
git clone https://github.com/Maleesha20010203/WeatherAppUsingReact.git
cd WeatherAppUsingReact
npm install
npm start
